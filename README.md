# dhall-vim

This package provides syntax highlighting for
[Dhall](https://hackage.haskell.org/package/dhall), a configuration language.

## Installation

Using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'vmchale/dhall-vim'
```
